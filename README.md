# My Property System

## Overview

My Property System is a web-based property management application designed to simplify property administration for landlords, tenants, and property managers. The system centralizes property records, tenant information, rent payments, maintenance requests, and reporting in one easy-to-use platform.

## Features

- Secure user authentication
- Property management
- Tenant management
- Rent payment tracking
- Maintenance request management
- Dashboard with analytics
- Responsive user interface
- Database-driven architecture

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap

## Installation

1. Clone the repository
2. Import the SQL database into MySQL
3. Configure the database connection
4. Place the project in your XAMPP `htdocs` folder
5. Start Apache and MySQL
6. Open the application in your browser
